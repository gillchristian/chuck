# chuck

## Development

Install all dependencies:

```
$ yarn
```

Run development server:

```
$ yarn start
```

Build for production:

```
$ yarn build
```

Run tests:

```
$ yarn test
```

Run tests in watch mode:

```
$ yarn test:watch
```

In a separate terminal run (unless `yarn start` is already running):

```
$ yarn watch-reason
```
